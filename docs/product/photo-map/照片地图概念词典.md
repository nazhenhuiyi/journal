# 照片地图概念词典

这份文档用于统一照片地图讨论里的名词。它只做短解释；字段和对象关系见 [照片地图数据模型](照片地图数据模型.md)，完整流程见 [照片地图功能总览](照片地图功能总览.md) 和 [照片地图数据流与渲染](照片地图数据流与渲染.md)。

## 1. 内容与数据概念

| 概念 | 是什么 | 不是 |
| --- | --- | --- |
| `MurmurBlock` | 日记里的碎碎念 block，可能有正文、时间、位置和图片 | 不是地图 marker 本身 |
| `ImageBlock` | 日记里的图片 block，可能有原图路径、尺寸、EXIF 位置 | 不是缩略图缓存 |
| `PhotoMapMurmurSlice` | 当前筛选周期内参与照片地图计算的 murmur 临时切片 | 不是 marker，也不是持久数据 |
| `PhotoMapImageObservation` | 图片层的最终空间对象，一张图片一个 observation | 不是一条 murmur 的代表图 |
| `PhotoMapTextObservation` | 文字层的最终空间对象，一条有正文碎碎念一个 observation | 不是图片说明文字 |
| `coordinateSource` | 标记图片 observation 的坐标来自 `image` 还是 fallback 的 `murmur` | 不表示坐标一定来自真实 EXIF |
| 可用坐标 | finite、范围合法、且不是 `0,0` 的坐标 | 不包括 `NaN`、`Infinity`、越界和 `0,0` |
| 未定位内容 | 当前筛选周期内没有可用坐标的内容 | 不是数据丢失；只是不上地图 |

## 2. 聚合概念

| 概念 | 是什么 | 不是 |
| --- | --- | --- |
| cluster | 150m 内相邻 observation 的运行时分组 | 不是日记数据结构 |
| `PhotoMapImageCluster` | 图片 observation 的聚合组 | 不包含文字 observation |
| `PhotoMapTextCluster` | 文字 observation 的聚合组 | 不包含图片 observation |
| `representativeItem` | cluster 的代表项，用来展示代表图或代表文字 | 不代表用户选中了这项 |
| cluster center | 组内 observation 坐标平均值 | 不是任一条原始内容的权威坐标 |
| 链式相邻 | A-B 在 150m 内、B-C 在 150m 内时，A/B/C 进入同一组 | 不是只按质心距离聚合 |
| bridge observation | 一个新 observation 同时连上多个组时，把这些组并起来 | 不是重复显示的点 |
| visible cluster limit | marker 过多时只展示一部分，但保留 active/selected 组 | 不是过滤掉日记内容 |

## 3. 地图与 UI 概念

| 概念 | 是什么 | 不是 |
| --- | --- | --- |
| image marker | 地图上的图片点；单图是缩略图，聚合是代表缩略图加 `N张` | 不是底部文字卡片 |
| text marker | 地图上的文字点；单条是小绿点，聚合是数字点 | 不是图片 marker 的附属 |
| expanded image marker | 图片组展开后的单张环形缩略图 | 不是新的日记内容 |
| expanded text marker | 文字组展开后的小绿点 | 不是 sheet 条目本身 |
| 照片 tray | 图片组展开后的底部照片列表 | 不是文字 sheet |
| 附近碎碎念 sheet | 文字组展开后的附近内容列表 | 不是底部 murmur 卡片 |
| 底部 murmur 卡片 | 当前 `selectedTextId` 对应的文字摘要卡片 | 不是 cluster sheet |
| 图片 preview | 全屏查看原图或 gallery | 不负责改变文字选择 |
| 当天只读页 | 从卡片主体或 sheet 条目进入的日期阅读页 | 不是地图内 sheet |
| 顶部统计卡 | 当前筛选周期的数量概览 | 不是筛选器本身 |
| 路线 | 按 murmur 本体坐标画出的线 | 不使用图片坐标补线 |

## 4. 状态概念

| 概念 | 是什么 | 不是 |
| --- | --- | --- |
| `selectedTextId` | 当前底部 murmur 卡片对应的文字 observation id | 不控制图片组展开 |
| `interaction` | 地图临时交互焦点，可能是 `browse`、`imageCluster` 或 `textCluster` | 不是持久业务数据 |
| `browse` | 普通地图浏览状态 | 不显示 tray、sheet 或展开 marker |
| `imageCluster` | 图片组展开状态 | 不改变 `selectedTextId` |
| `textCluster` | 文字组展开状态 | 打开附近 sheet，并隐藏底部 murmur 卡片 |
| `motion: enter` | 用户主动进入某个 cluster，用于播放入场动画和相机动画 | 不是恢复现场 |
| `motion: restore` | 从 preview 返回前恢复图片组现场 | 不应该重播展开动画 |
| `revision` | 标记本次 interaction 进入次数，用来控制动画是否需要重播 | 不是数据版本号 |
| snapshot | 页面级会话快照，保存 `range`、`interaction` 和 `selectedTextId`，用于从只读页返回照片地图 | 不是永久缓存 |

## 5. 技术概念

| 概念 | 是什么 | 不是 |
| --- | --- | --- |
| `MapLibreMap` | MapLibre React Native 地图容器 | 不承载日记业务状态 |
| `Camera` | 地图视角控制器，负责移动、缩放和 fitBounds | 不是用户选择状态 |
| `GeoJSONSource` | MapLibre 的地图数据源，适合批量渲染点线面 | 当前不用于图片 marker |
| `Layer` | MapLibre style layer，把 source 画成线、圆点或 symbol | 当前只用于路线 |
| `Marker` | 把 React Native view 放到地图坐标上的 MapLibre 组件 | 不是普通页面里的纯 View 冒泡链 |
| `event.stopPropagation()` | marker/source 点击阻止继续冒泡到 `Map.onPress` | 不是业务状态 |
| `guardOverlayMapPress()` | overlay/marker 点击后的短窗口兜底，防止误触发地图空白点击 | 不是主要交互机制 |
| thumbnail cache | 应用缓存目录里的地图小图 | 不是日记内容，也不同步 |
| EXIF fallback | Android 相册读取位置失败时，通过 MediaLibrary 再尝试读取 asset 位置 | 不是用当前系统定位补图片位置 |

## 6. 常见说法对照

| 口头说法 | 精确说法 |
| --- | --- |
| “这个图片点” | 一个 `PhotoMapImageObservation` 渲染出的 image marker |
| “这个文字点” | 一个 `PhotoMapTextObservation` 渲染出的 text marker |
| “附近 6 条” | 当前文字 observation 所在 `PhotoMapTextCluster` 的 item 数 |
| “展开 group” | `interaction` 进入 `imageCluster` 或 `textCluster` |
| “收起 group” | 清掉 `interaction`，回到 `browse` |
| “打开那天” | 导航到对应日期的只读页 |
| “图片位置不准” | 先区分 `coordinateSource` 是 `image` 还是 `murmur` |
| “地图点没出现” | 先检查坐标是否可用、筛选周期是否包含、是否超过可见 cluster limit |
