# 设计资产

这里放且留的视觉探索、logo 过程稿、导出图和产品概念图。它是设计资产库，不是产品决策文档入口。

## 当前内容

- `qieliucike-*`：且留 logo、印章、取景框、配色和图像生成探索过程稿。
- `logo-exports/`、`logo-exports-tight/`：不同尺寸、透明背景和暖色背景的 logo 导出图。
- `svg-*-quicklook-preview/`：SVG 优化和描摹效果的预览图。
- `old-days-*`：旧日回看和机制展示的视觉概念。
- `ribbon-pull-*`：卡片、明信片和产品动效方向的视觉概念。
- `all-pages-home-mockup.svg`、`handdrawn-icon-directions.svg`：页面和手绘图标方向稿。

## 维护边界

- 产品定位、文案、功能边界放到 [产品文档](../product/README.md)。
- 官网可公开内容放到 [对外内容](../public/README.md)。
- 当前目录可以保留过程资产；只有进入产品或官网的最终资源，才需要同步到对应 app/site 的 `public` 或 assets 目录。
